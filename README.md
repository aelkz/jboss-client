# jboss-client

AngularJS application frontend for a hosted jboss REST api

## Build and Run
In the jdv-map directory

`npm install`

`bower install`

`grunt client`

## Editing
The proxy that allows the cross domain query is located in
'Gruntfile.js'

By default the context is on a server 'demo-vm' on port 8080
